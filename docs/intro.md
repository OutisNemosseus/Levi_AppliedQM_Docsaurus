## Getting Started

To run these programs:
1. Download the `.m` file from any program page
2. Open MATLAB
3. Set the MATLAB path to the directory containing the downloaded file
4. Type the filename (without `.m` extension) in the MATLAB command window
5. Press Enter to execute

Some programs will prompt for keyboard input, and most display results graphically with intermediate results in the MATLAB command window.
