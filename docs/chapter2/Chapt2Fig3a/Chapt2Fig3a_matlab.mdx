## Source Code

```matlab title="Chapt2Fig3a.m"
%Chapt2Fig3a.m%200 THz electric field modulated by Gaussian envelope;clear;clf;tau0=14.14e-15;							%time constant (s)tau02=tau0^2;w0=2*pi*200.e12;							%200 THz optical frequency (rad s-1)t0=0.5e-13;									%time t0 (s)npoints=500;								%number of points plottedtmax=1.e-13;								%maximum time (s)tstep=tmax/npoints						%time step (s)t=[0:tstep:tmax];							%time [start:increment:stop]   y=cos(w0*t);y=y.*exp(-(t-t0).^2/tau02);figure(1);plot(t,y,'b');								%electric field as function of timeaxis([0,tmax,-1,1]);temp2=sprintf('Chapt2Fig3a, w0 = %5.3e rad s-1, tau0 = %5.3e s',w0,tau0);title(temp2);xlabel(['Time, t (s)']);ylabel(['Amplitude']);
```
