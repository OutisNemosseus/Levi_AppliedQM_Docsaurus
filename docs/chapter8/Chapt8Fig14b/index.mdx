\begin{abstract}
This document analyzes \texttt{Chapt8Fig14b.m}, which plots the differential scattering rate as a function of scattering angle using Thomas-Fermi screening. Unlike Chapt8Fig14a (which integrates over angles), this code shows the angular distribution, revealing the forward-scattering peak characteristic of screened Coulomb scattering.
\end{abstract}

%==============================================================================
\section{Theoretical Foundation}
%==============================================================================

\subsection{Differential Scattering Rate}

\begin{definition}[Angular Differential Rate]
The differential scattering rate per unit angle is:
\begin{equation}
\frac{d(1/\tau)}{d\theta} = \frac{2\pi m^*}{\hbar^3 k^3} n_i \left(\frac{e^2}{4\pi\varepsilon_0\varepsilon_r}\right)^2 \frac{\sin\theta}{(q^2 + q_{\text{TF}}^2)^2}
\end{equation}
where $q = 2k\sin(\theta/2)$.
\end{definition}

\subsection{Forward Scattering Peak}

\begin{theorem}[Small-Angle Dominance]
At small angles ($\theta \to 0$):
\begin{equation}
q = 2k\sin(\theta/2) \approx k\theta \to 0
\end{equation}
Without screening, the rate diverges as $1/q^4 \propto 1/\theta^4$.

With Thomas-Fermi screening:
\begin{equation}
\frac{d(1/\tau)}{d\theta} \propto \frac{\sin\theta}{(k^2\theta^2 + q_{\text{TF}}^2)^2}
\end{equation}
The screening regularizes the divergence, producing a finite peak at small $\theta$.
\end{theorem}

\subsection{Transformation to $\eta$ Variable}

Using $\eta = \sin(\theta/2)$:
\begin{equation}
\frac{d(1/\tau)}{d\theta} = \frac{d(1/\tau)}{d\eta} \cdot \frac{d\eta}{d\theta} = \frac{d(1/\tau)}{d\eta} \cdot \frac{\cos(\theta/2)}{2}
\end{equation}

%==============================================================================
\section{Physical Interpretation}
%==============================================================================

\subsection{Forward Scattering Peak}

The plot shows:
\begin{itemize}
\item Peak at small angles (forward scattering dominates)
\item Rapid decay with increasing angle
\item Higher energy electrons have broader angular distribution
\end{itemize}

\subsection{Effect of Screening}

Without screening ($q_{\text{TF}} = 0$):
\begin{equation}
\frac{d(1/\tau)}{d\theta} \propto \frac{1}{\sin^4(\theta/2)} \to \infty \text{ as } \theta \to 0
\end{equation}

With Thomas-Fermi screening:
\begin{equation}
\frac{d(1/\tau)}{d\theta} \propto \frac{1}{(\sin^2(\theta/2) + q_{\text{TF}}^2/4k^2)^2}
\end{equation}

The screening cuts off the divergence at $\theta \sim q_{\text{TF}}/k$.

\subsection{Energy Dependence}

For $E = 0.1$ eV vs $E = 0.3$ eV:
\begin{itemize}
\item Higher energy $\Rightarrow$ larger $k$
\item Larger $k$ $\Rightarrow$ smaller peak angle (screening less effective)
\item Overall rate lower at higher energy ($\propto 1/k^3$)
\end{itemize}

\section{Summary}
%==============================================================================

This code plots the angular distribution of the ionized impurity scattering rate, showing the characteristic forward-scattering peak that is regularized by Thomas-Fermi screening. The peak width is determined by the ratio $q_{\text{TF}}/k$, and the amplitude scales inversely with $k^3$. This complements Chapt8Fig14a, which integrates over angles to give the total scattering rate.
